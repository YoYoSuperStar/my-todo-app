# Simple To-Do App

A beginner-friendly Python project for managing a list of tasks.

## Files

- `todo.py` — the main app logic (add, complete, and list tasks)

## How to run

```
python todo.py
```

## What it does

1. Adds a few sample tasks
2. Displays the full task list
3. Marks one task as completed
4. Displays the updated list
